# A-Void
#### A spaceship game

The goal of this game is to 'A-Void' blocks by making your ship go right or left.

## Install

First, set `block.png` and `ship.png` into `assets/` directory.

```shell
pip3 install -U pip
pip3 install kivy
```

## Play

Then, you can launch:

```shell
python3 main.py
```
